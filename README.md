# Привет, я Данила 👋

**Автоматизирую бизнес кодом.**

В разработку я пришёл не через учебники по фреймворкам, а через работу: руководил IT-отделом, внедрял CRM, с нуля запускал складской учёт. В какой-то момент настраивать чужие инструменты стало тесно — захотелось делать свои. Поэтому мои проекты начинаются не с «хочу пощупать новую библиотеку», а с «есть процесс, который отнимает время, — сейчас автоматизирую».

## Чем занимаюсь

- Автоматизирую бизнес-процессы: от документооборота до мониторинга оборудования.
- Пишу веб-приложения и бэкенды на Python.
- Перевожу с языка бизнеса на язык технического задания и обратно.
- Быстро вникаю в новое: удалённый мониторинг зарядных станций освоил за пару недель, потому что этого требовала работа.

## Стек

- **Языки:** Python, JavaScript, C++ (базовый уровень).
- **Веб:** HTML, CSS, JavaScript; Tilda, Wix.
- **Бизнес-инструменты:** 1С, CRM, ЭДО (Контур, СБИС).
- **Протоколы и оборудование:** OCPP (зарядные станции для электромобилей).
- **Языки:** русский — родной, английский — B2.
